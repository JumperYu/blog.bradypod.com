linux上搭建vip负载均衡 2015.03.09

lvs (Linux Virtual Server)

LVS 是国内的一位牛人发起的开源项目，自Linux2.4之后被收录进去了， 非常值得学习。

我们需要结合keepalived + vip来实现lvs， 以下是安装过程， 真正繁琐的是在配置。 

cat /etc/issue    # 这是我的操作系统版本
CentOS release 6.5 (Final)
uname -r   # 这是我的linux内核版本
2.6.32-431.23.3.el6.x86_64

# 源码安装
cd /usr/local/src
wget www.linuxvirtualserver.org/software/kernel-2.6/ipvsadm-1.26.tar.gz
tar zxf ipvsadm-1.26.tar.gz
cd ipvsadm-1.26 &&  make && make install  # 可能会报错
# yum -y install popt popt-devel libnl libnl-devel popt-static openssl-devel  #如果还没有这些依赖先安装这些
wget www.keepalived.org/software/keepalived-1.2.15.tar.gz
tar zxf keepalived-1.2.15.tar.gz && cd keepalived-1.2.15 && ./configure && make && make install

# 由于我是centos系统， 所以我直接用yum安装这2货 （:
yum install ipvsadm -y
yum install keepalived -y
# 确实这样简单 (:(:
